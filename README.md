# Rolependium - Tu Grimorio Digital para D&D


Rolependium es una aplicación Android diseñada para jugadores y Dungeon Masters de Dungeons & Dragons 5ª edición. La app facilita la gestión de personajes, hechizos y otras herramientas esenciales para tus aventuras.

Dentro del branch master de encuentra el proyecto de AndroidStudio de la aplicacion y en esta branch main se almacena lo referente a la creacion del fichero csv además de un enlace de descarga al fichero apk de la aplicacion.
